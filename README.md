# Simple To-Do List

A clean, lightweight to-do list application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add, complete, and delete tasks
- Task persistence using browser localStorage
- Task statistics (total and completed counts)
- Mobile-responsive design
- No dependencies or build process required

## Live Demo

Deployed on Vercel: [Coming soon after deployment]

## Local Usage

Simply open `index.html` in your web browser. No server or build process required.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API

## Deployment

This project is deployed to Vercel via GitHub integration. Any push to the main branch automatically triggers a new deployment.
